# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/TiagoSilva-dev/Sistema-de-Chamados.git && cd Sistema-de-Chamados
# Install Dependencies
$ yarn install
# Run Aplication
$ yarn start
```
